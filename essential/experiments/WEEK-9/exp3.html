Responsive screen for contact page

About the experiment

In this experiment, I have made my contact page responsive to different screen sizes making it easily accessible on different screens.

Code Snippet

@media ( min-width : 1200px) .container {
	width:1170px;
}
@media ( min-width : 992px) .container {
	width:970px;
}
@media ( min-width : 768px) .container {
	width:750px;
}
@media screen and ( max-width : 992px) {
	h1 {
		line-height: 55px;
		font-size: 80px;
		padding-top: 60px;
		margin-left: 15px;
	}
	.navbar-collapse {
		box-shadow: none;
	}
	
	.map-image{
	     padding: auto;
	}
}
@media screen and (max-width : 768px){
  .nav > li > a{
       border-bottom:1px solid #f1f4f7;
       line-height: 20px;
  }
  .logo-img{
    margin-left:-30px;
    width:65px;
    height:65px;
  }
  
  .navbar-toggle{
    margin-right: 5px;
  }
  
  .collapse navbar-collapse{
    border-left-width: -10px;
  }
  
  	.map-image{
	    padding: auto;
	}
}
