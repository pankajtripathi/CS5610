Drag me around

Drop here

Code Snippet

In the code below I have created two boxes draggableBox and droppableBox. The draggableBox when dragged and dropped within the droppablBox the text content of the droppable box is changed from Drop here to dropped, this to make sure that the draggableBox has been dropped within the droppableBox

 $("#draggableBox").css({
  "width": "100px",
  "height": "100px",
  "padding": "0.5em",
  "background-color": "green",
  "float":"left"
 });
 $("#droppableBox").css({
  "width": "150px",
  "height": "150px",
  "padding": "0.5em",
  "clear": "both",
  "background-color": "blue",
  "margin": "10px"
 });
 $("#draggableBox").draggable();
 $("#droppableBox").droppable({
  drop: function (event, ui) {
  $(this)
  .addClass("ui-state-highlight")
  .find("p")
  .html("Dropped!");
  }
 });

Description

The above example illustrates draggable and droppable boxes.

By using the functions draggable and droppable two boxes are assigned respective properties. When the dragableBox is dropped in the droppableBox the content of the droppableBox is changed using code .html("Dropped!")
