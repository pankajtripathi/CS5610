jQuery code

In the code given below I have created a selectable list using jQuery function .selectable() On selecting an item from the list the color of the list item is changed from #selectable .ui-selecting { background: #FECA40; to #selectable .ui-selected { background: #F39814; color: white; }

 $("#selectable").css({
  "list-style-type": "none",
  "margin": "0",
  "padding": "0",
  "width": "60%"
 });
 $("selectable li").css({
  "margin": "3px",
  "pading": "0.4em",
  "font-size": "1.4em",
  "height": "18px"
 });
  $("#selectable").selectable();

CSS code

  #selectable .ui-selecting { background: #FECA40; }
  #selectable .ui-selected { background: #F39814; color: white; }

Description

The above example illustrates selectable elements using jquery. In the above example using jQuery funtion .selectable we can create a selectable list of items.

When an item in the list is selected we can change its color to any corresponding color to dufferentiate between a selected and unselected list item.
