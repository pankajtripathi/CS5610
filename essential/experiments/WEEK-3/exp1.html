In this experiment I have writeen a jQuery function using which I have made changes in the styling of header tag H1

Along with creating CSS for header tag I have also made changes in CSS for paragraphs wherein I have changes their backgorund color and font color

I have used append function to add some text before second paragraph and prepend to add some text before first paragragh

  $(function () {
 $("h1").css({
 "color": "red",
 "background-color": "yellow"
 });
 $("p#second").css({
 "color": "green"
 });
 $("p.first").css({
 "background-color": "lightblue"
 });
 $(".whiteBg").css({
 "color": "white"
 });
 $("p#second").append("Added new content using append");
 $("p#second").append("<p>This is new paagraph added via jQuery</p>");
 var newParagraph = $("<p>This is new paragraph added with a variable</p>");
 $("p.first").prepend(newParagraph);
 newParagraph.css({ "color": "red" });
  });

Description

The above example illustrates how using jQuery we can style our web content. In the above experiment using jQuery function I have changed CSS properties for some tags like h1, p#second, p.first.

Using perpend and append functions data can be added to the existing page which has been illustrated successfully in the above example
