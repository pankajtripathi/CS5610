CSS Experiments

Image- Hover effect


Code Snippet

 img{
  opacity:1.0
  filter:alpha(opacity:100)
 }
 img:hover{
  opacity:0.4;
  filter:alpha(opacity=50);
 }

Description

This example shows the effect of hovering on an image.

The image turns transparent when mouse is hovered it

Transition

Code Snippet

 .transition{
  width:100px;
  height:100px;
  background-color:blue;
  -webkit-transition: width 2s;
  transition: width 2s;
 }
 .transition:hover{
  width:200px;
 }

Description

The example above show CSS transition property

The transition takes place when mouse is hovered to the object
