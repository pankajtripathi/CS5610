About the experiment

In this experiment, I have created the homepage for my project which basically has alogo which is url to homepage and sone text on the page giving info about the web site In this experiment I have added a bootstrap toggle button to show and hide content

Code Snippet

    $(function(){
	$('#want-more').click(function() {
	    if ( $('#want-more-text').is(':hidden')) {
	      $('#want-more-text').slideDown('slow');
	      $(this).text('SHOW LESS');
	    } else {
	      $('#want-more-text').slideUp();
	      $(this).text('SHOW MORE');
	    } 
	  });
});
