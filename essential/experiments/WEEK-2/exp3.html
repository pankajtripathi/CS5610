Code Snippet

 @media screen and (max-width: 500px)
  {
  body {
  background-color:red;
  }
 }
 @media screen and (min-width: 500px) and (max-width:700px)
  {
  body {
  background-color:green;
  }
  .container {
  width:400px;
  margin:auto;
  }
  .row-1,.row-2{
  width:400px;
  clear:both;
  }
  .col-6{
  width:200px;
  float:left;
  }
 }
 @media screen and (min-width: 700px)
  {
  body {
  background-color:yellow;
  }
  .container {
  width:600px;
  margin:auto;
  }
  .row-1,.row-2{
  width:600px;
  clear:both;
  }
  .col-6{
  width:300px;
  float:left;
  }
 }

Description

The above example illustrates a responsive grid with a responsive container

As the screen size changes the columns are displayed as per the row size

If the row size is 600px then the column size is half of that that is 300px
