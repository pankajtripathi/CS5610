Responsive Form

Code Snippet

 @media screen and (max-width: 500px)
  {
  body {
  background-color:red;
 }
 #firstName, #lastName, #email, #pwd {
  float:left;
  }
 }
 @media screen and (min-width: 700px)
  {
  body {
  background-color:yellow;
 }
 .container {
  width:600px;
  margin:auto;
  }
 }
 @media screen and (min-width: 500px) and (max-width:700px)
  {
  body {
  background-color:green;
 }
 .container {
  width:300px;
  margin:auto;
  }
 }

Description

The above example illustrates a responsive form with a responsive container

The container and the form elements align as per the screen size
