In this code I have created an app call MyNoteApp. I am saving my text in $scope.message. I am showing the characters left by deducting the text entered with permissible limit of 100 characters. I have provided a functionality to save the note with the date. I have added the title of the note and the date it was created in an array called notes which I am dispalying on the page

  app = angular.module("MyNoteApp", []);
 app.controller("myNoteCtrl", function ($scope) {
  $scope.notes = [];
  $scope.message = "";
  $scope.left = function () {
   var rem = 100 - $scope.message.length;
   return rem;
  };
  $scope.clear = function ()
  {
   $scope.message = "";
  };
  $scope.save = function () {
   $scope.currentDate = new Date();
   var note = {
   title: $scope.message,
   date:$scope.currentDate
  }
  $scope.notes.push(note);
  };
 });

AngularJS, commonly referred to as Angular, is an open-source web application framework maintained by Google and a community of individual developers and corporations to address many of the challenges encountered in developing single-page applications. Its goal is to simplify both development and testing of such applications by providing a framework for client-side model–view–controller (MVC) architecture, along with components commonly used in rich internet applications.
