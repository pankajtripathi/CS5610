In this code using controller to display on the webpage the movies directly from the server based on the title provided by the user. The movies in the list can removed dynamically or a new movie can be added to list by the user. To add a new movie user has to click on "+" and to remove he/she has to click on "X". To remove the movie I obtain the idex of the movie for which the button is clicked and I just splice it from the list. To add a movie I obtain the details from the view and push it to list of movies.I have also provided a functionality to add a movie to a favorites list.
