MovieTime - Project Documentation



Acknowledgment:

Goal:

The objective of this web development project is to design a website where user can check out the movies currently in theaters, upcoming movies, movies currently topping box office charts, search movies and along with these features user can login and create his own list of movies to watch.

System Features:

1. Search Movie

This feature allows user to search for hisher favorite movie from the RottenTomatoes API and check the movie details.

2. In Theaters

This feature provides user with all the movies currently showing in theaters that he/she can check out. User can click on the result and check the movie details

3. Box Office

This feature provides user with movies currently topping box office charts. User can click on the result and check the movie details

4. Opening Movies

This feature allows user to check out movies opening this week. User can click on the result and check the movie details

5. Upcoming movies

This feature allows user to check out movies coming next week and month. User can click on the result and check the movie details

6. Movie Reviews

This feature allows user to check out movie reviews on the movie details page.

7. Login

This feature enables user to login and create his/her own list of movies to watch
